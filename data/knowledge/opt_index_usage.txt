L'optimisation des index est cruciale. Utilisez un index B-Tree pour les colonnes à forte sélectivité (beaucoup de valeurs uniques). Utilisez un index Bitmap pour les colonnes à faible sélectivité comme le genre ou le pays. Si une requête affiche "TABLE ACCESS FULL" sur une grande table, un index est probablement manquant ou ignoré à cause d'une fonction SQL sur la colonne filtrée.
